# Scripts

This directory contains curated public-facing scripts.

- `train/`: smoke/pipeline-compatible training launchers.
- `evaluate/`: summary/evaluation scripts.
- `figures/`: selected figure-generation scripts used by the paper.
- `preprocess/`: preprocessing helpers.
- `configs/`: historical cluster job references documenting the manuscript runs.

General one-off OOF export scripts, t-SNE sweeps and cache builders are not included in the GitHub package because they are either redundant with checkpoints or were in-house workflow utilities.
