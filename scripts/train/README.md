# Training Entry Points

- `train_bcdb_smoke.sh`: verifies that the BCDB CLI imports and exposes arguments.
- `train_md_final1640_v2_smoke.sh`: verifies that the MD CLI imports and exposes arguments.
- `run_bcdb_pipeline.sh`: direct BCDB/homopolymer pipeline runner.
- `run_md_pipeline.sh`: direct MD_FINAL1640_v2 pipeline runner.

Full manuscript-scale training requires processed caches/checkpoint assets and cluster resources. Historical command templates are retained under `scripts/configs/`.
